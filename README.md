# NIDS — Network Intrusion Detection System

## Requirements

- Python 3.11
- CICIDS2017 dataset CSV files placed in `data/raw/` (download from https://www.unb.ca/cic/datasets/ids-2017.html)

---

## Setup

```bash
git clone https://github.com/ayorex11/nids.git
cd nids

python -m venv venv
source venv/bin/activate        # Windows: .\venv\Scripts\activate

pip install -r requirements.txt
```

---

## Running

Run each script in this order:

```bash
python src/preprocess.py
python src/feature_selection.py
python src/train_rf.py
python src/train_dnn.py
python src/train_lstm.py
python src/train_dqn.py
python src/evaluate.py
```

Then launch the app:

```bash
streamlit run app/streamlit_app.py
```

---

## Notes

- Training was done on CPU. RF is fast. DNN and LSTM take 15-35 minutes each. DQN takes 2-3 hours. GPU will be faster.
- Do not skip steps, each one depends on the previous.
- The app accepts any CICIDS2017-format CSV file generated by CICFlowMeter.
